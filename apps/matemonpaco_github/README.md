# Batallas MateMon Paco - versión estática

Versión para GitHub Pages sin Lovable y sin Supabase.

## Archivos
- index.html: pantalla del juego.
- styles.css: estilos.
- app.js: lógica del combate.
- data/cards.json: datos de las 53 cartas.
- data/effectiveness.json: efectividades.
- assets/cards/: imágenes.

## Publicar
Sube el contenido completo de esta carpeta a un repositorio y activa GitHub Pages en Settings > Pages, rama main, carpeta /root.

## Modificar
Edita data/cards.json para cambiar cartas, ataques, PS, velocidad o tipos. Sustituye imágenes en assets/cards conservando el nombre del archivo.
