# Potencia Pop

Juego educativo tipo Puzzle Bobble para practicar cuadrados y cubos.

## Archivos
- `index.html`: estructura de pantallas.
- `style.css`: estilos visuales y animaciones UI.
- `game.js`: motor del juego, modos, niveles, sonidos, progreso y logros.

## Uso
Sube la carpeta completa a GitHub Pages o abre `index.html` en el navegador.
El progreso se guarda en `localStorage` del navegador.
