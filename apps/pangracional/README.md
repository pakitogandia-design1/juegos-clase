# Pang Racional

Juego educativo arcade en Phaser 3 inspirado en la jugabilidad clásica de Pang/Buster Bros, pero con estética, código y assets generativos propios.

## Cómo usarlo

1. Sube la carpeta completa a GitHub.
2. Activa GitHub Pages en la rama donde esté `index.html`.
3. Abre la URL publicada.

No necesita instalación ni npm. Phaser carga desde CDN.

## Controles

### Ordenador
- Flechas o A/D: mover
- Arriba/abajo: subir y bajar por escaleras
- Espacio: disparar
- Shift, Tab o C: cambiar arpón
- 1: arpón racional
- 2: arpón irracional
- P o Esc: pausa

### Móvil
Incluye controles táctiles en pantalla.

## Reglas matemáticas

- Arpón racional destruye números racionales.
- Arpón irracional destruye números irracionales.
- Las burbujas no dan pistas por color: el color es estético y no indica categoría.
- Los decimales finitos y periódicos son racionales.
- Los decimales infinitos no periódicos son irracionales.
- No se incluyen casos trampa de raíces cuadradas perfectas escondidas.
