# Identificando Material Vegetal 4.0

Juego didáctico web listo para subir a GitHub Pages o abrir localmente.

## Contenido
- Menú principal con tarjetas, estrellas y progreso.
- Guardado del progreso en el navegador.
- Pantalla de progreso y pantalla final de tema superado.
- Juegos específicos para:
  - Partes de la flor (usando la imagen del tema)
  - Partes de la hoja (usando la imagen del tema adaptada, sin mostrar el concepto dentro de cada recuadro)

## Uso
1. Descomprime el archivo.
2. Abre `index.html` en el navegador, o súbelo a GitHub.
3. Si se publica en GitHub Pages, no necesita instalación adicional.
